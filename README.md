##NavBar
> 可提供成功、失败、警告、错误等提示信息，是一种不打断用户操作的轻量级提示方式。


| 属性             |      说明          |           类型              |       默认值     |
|-----------------|-------------------|----------------------------|:----------------:|
|prefixCls|样式前缀，如：`cefc-mesTemp`，可用于自定义样式|String          |'cefc-mesTemp'|
|status|消息状态，可取值为succ/fail/waring/error| String |   ''      |
|msgResult|     消息结果 |  String      |   ''  |
|resuCause|     结果原因    |   String  |   ''  |
